==== BLOG MODULE INSTALLATION GUIDE - Version 1.1.8 ===
1, Unzip (decompress) the module archive file.
2, Copy folder plblog into your PrestaShop /modules folder.
2, Go to Admin-> modules tab.
3, Click button install for installing module at tab Front Office Features
4, Click Configure for managing module's configuration
5, Click tab Tools-> Blog for manage categories, posts, comments, tags
6, If you want to use URL Friendly, firstly backup your current .htaccess then go to
Back-Office-> Preferences-> SEO&URLs then generate .htaccess file again
or go to Tool-> Generators
You can see more detail how to use URL-friendly and generate .htacess file here
http://www.prestashopdocumentation.info/how-to-enable-friendly-url-in-prestashop-1-4/
