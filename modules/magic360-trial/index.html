What's new in v2.1?

Lots of features requested by users are now available:

  • Change text on hover over buttons (example).
  • API gives more control to start, stop, load spins (example).
  • New JavaScript callback "onready" (example).
  • Default auto-spin speed reduced.
  • Resume infinite spin on click (example).
  • Resume infinite spin on click of the spin button.
  • Resume infinite spin on mouseout (pause on hover) (example).
  • Stop auto-spin on drag.
  • Hide buttons on non-stop-spin (example).
  • Mousewheel ignored during auto-spin.
  • Non-stop-spin cannot be paused by drag or scrollwheel (example).

What's new in v2.0?

Magic 360™ version 2.0 has been completely rewritten in JavaScript with many new features:

  • Works on iPad, iPhone, Android and other mobile devices.
  • Multiple rows/angles to rotate products up/down as well as left/right (example).
  • Auto-spin once, twice or infinitely.
  • Button to turn zoom on/off.
  • Click to magnify (example).
  • Faster load time.
  • Simplified setup.
  • Customizable buttons.
  • Customizable loading message.
  • Fetch images on page-load, hover or click.
  • Use any number of images - from 2 to 72 (or even more!).
  • More examples online.
